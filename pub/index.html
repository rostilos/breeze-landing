01. Can I use Breeze with my Luma-based theme?

Yes, you can install the swissup/breeze module separately to use Breeze with your Luma-based theme.

02. Will my current modules work with Breeze?

Modules that don't rely on JS will work with Breeze. However, modules that utilize JS may require integration efforts to function correctly

03. How do I integrate my modules with Breeze?

We recommend hiring a professional developer for this task. You can also refer to our integration tips available in the documentation.

04. How can Perspective help me with Breeze?

Perspective's certified Magento developers can provide fast installation and support for Breeze, ensuring a smooth integration with your existing store. Their expertise will help you take full advantage of Breeze's features and performance improvements.

05. How does Breeze compare to the Hyva theme for Magento 2?

Breeze focuses on reducing JS stack complexity and script evaluation time, while Hyva emphasizes a minimalistic and clean frontend approach. If you're unsure which theme is right for your store, consult Perspective's expert developers to help you make the best decision.
